Function Details

run/4

run(G::graph:graph(), S::graph:vertex(), T::graph:vertex(), Mode::mode()) -> graph_lib:flow() | {error, not_network}

Runs the Edmonds-Karp or Ford-Fulkerson algorithm on a graph G with S as source as T as sink.

When Mode is dfs the algorithm is called Ford-Fulkerson and when Mode is bfs the algorithm is called Edmonds-Karp.
